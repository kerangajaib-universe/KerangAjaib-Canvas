Buku Panduan

Mendapatkan API Key

Langkah pertama untuk menghubungkan otak AI.

Tools ini menggunakan AI Google Gemini untuk berpikir. Anda memerlukan "Kunci Akses" (API Key) gratis dari Google.

  1. Buka situs Google AI Studio ↗ dan login dengan akun Google/Gmail Anda.
  2. Klik tombol biru "Create API key".
  3. Salin (Copy) deretan teks acak yang panjang tersebut.
  4. Kembali ke tools Storyboard ini, paste key tersebut di kolom "API Key" pada menu kiri.
  5. Klik tombol Test. Jika berhasil, daftar model AI akan muncul.

Form & Eksekusi

Meracik ide dan parameter cerita ajaib Anda.

Setelah API terkoneksi, saatnya menuangkan ide cerita Anda di panel sebelah kiri.

  • 💡
    Saran AI: Kalau Anda sedang buntu, klik tombol "Saran AI ✨". Aplikasi akan mengisikan ide cerita acak secara otomatis!
  • ✍️
    Karakter / Subjek: Isi bagian ini sedetail mungkin (misal: "Pria berjaket merah, rambut pirang"). Ini sangat penting agar AI menjaga tokoh utama Anda tetap konsisten di setiap scene.
  • 🎨
    Visual Style: Pilih gaya gambar kesukaan Anda (misal Anime atau Cinematic). AI akan meracik prompt khusus untuk menghasilkan gambar sesuai gaya tersebut.

Jika semua sudah diisi, klik tombol besar "Generate Storyboard" di paling bawah!

Text to Image

Mengubah naskah adegan menjadi foto visual pertama.

Sekarang Anda sudah punya naskah per *scene*. Langkah pertama adalah menyulap kotak teks "1. Visual Prompt (Image)" menjadi gambar diam (foto).

🛠️ Rekomendasi Tools Gratis:

1. Bing Image Creator (DALL-E 3)

Sangat pintar mengikuti prompt. Gratis token harian pakai akun Microsoft.

2. Leonardo.ai

Kualitas gambar sangat estetik dan cinematic. Gratis 150 token setiap hari.

Cara Pakai: Klik tombol "Copy" pada kotak nomor 1, lalu Paste ke web di atas. Simpan fotonya!

Image to Video

Menggerakkan foto diam menjadi video sinematik.

Bagian paling seru! Upload foto dari tahap sebelumnya, lalu gunakan kotak "2. Motion Prompt (Video)" untuk memberi perintah pergerakan pada AI Video.

🛠️ Rekomendasi Tools Video (Gratis):

1. Haiper.ai Paling Disarankan

Sepenuhnya gratis! Hasilnya sangat halus untuk durasi 2-4 detik.

2. Kling AI / Luma Dream Machine

Memberikan kredit gratis setiap hari. Sangat realistis untuk manusia.

Voiceover (Suara)

Menambahkan narasi dan dialog agar video lebih hidup.

Salin teks pada kotak "3. Voiceover / Narration" dan ubah menjadi suara dubbing profesional menggunakan AI.

🛠️ Rekomendasi Tools Suara:

1. ElevenLabs

Suara AI paling natural di dunia. Ada kuota gratis bulanan.

2. Fitur Text-to-Speech CapCut

Paling praktis karena langsung dilakukan di dalam aplikasi edit video.

Editing (Finalisasi)

Menyusun potongan video dan merender hasil akhir.

Semua bahan sudah siap! Potongan video pendek dari AI dan rekaman suara kini tinggal disatukan.

CapCut (Mobile / PC)

Aplikasi editing terbaik dan gratis. Susun video sesuai urutan SCENE. Masukkan audionya, tambahkan backsound, dan Mahakarya Anda siap di-upload! 🎉
